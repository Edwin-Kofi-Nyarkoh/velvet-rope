# Velvet Rope

A premium event ticketing and management platform for web and mobile.

## Stack

- **Web** — Next.js (TypeScript)
- **Mobile** — Expo / React Native (TypeScript)
- **API** — Node.js, Express, TypeScript
- **Database** — PostgreSQL via Prisma ORM
- **Payments** — Paystack

## Getting Started

```bash
pnpm install
```

Set up your environment variables (see each app's `.env` file), then:

```bash
pnpm dev          # web + API
pnpm dev:mobile   # mobile
```

## Local URLs

- Web: `http://localhost:3000`
- API: `http://localhost:4000/api`
